Tài liệu này chỉ hỗ trợ cho thiết bị di động.

Khởi động XAMPP
  • Mở XAMPP Control Panel
  • Click START cho Apache và MySQL (đèn xanh = OK)
Nếu Apache lỗi port: Config → httpd-ssl.conf → thêm # trước Listen 443 thành #Listen 443
  • Ctrl + S để Save
Cài Wordpress
  • Truy cập https://wordpress.org/download
  • Bấm Download Wordpress 7.0 để tải về
  • Chuột phải vào Wordpress chọn Extract here
  • Copy thư mục wordpress sau khi giải nén bỏ vào C:\xampp\htdocs
Tạo Database
  • Mở trình duyệt gõ localhost/phpmyadmin
  • Bấm New
  • Nhập tên database, ví dụ: noithat, chọn Create
Cài đặt WordPress
  • Truy cập http://localhost/wordpress → chọn ngôn ngữ
  • Bấm Continue sau đó bấm tiếp Let's go!
  • Nhập thông tin database như sau:
    • Database Name: noithat Username: root Password: (để trống) Host: localhost Table Prefix: wp_
  • Bấm Submit bấm Run the installation
  • Nhập tên site, username, password admin, email
  • Bấm install Wordpress sau đó bấm Login
Hoặc đăng nhập tại http://localhost/wordpress/wp-admin
